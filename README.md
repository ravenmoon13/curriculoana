# Página Web – Currículo Pessoal

Projeto de página web estática desenvolvido em HTML e CSS que apresenta um currículo profissional em formato digital.  
A proposta é criar uma página simples, elegante e responsiva que funcione como cartão de visita profissional na internet.

Esta página pode ser utilizada como base para portfólio pessoal, apresentação acadêmica ou currículo online.

---

## Demonstração

Após a publicação com GitHub Pages, o site poderá ser acessado em:

https://ramospresley.github.io/curriculoweb/

---

## Tecnologias utilizadas

- HTML5
- CSS3
- Layout Responsivo
- Font Awesome (ícones)
- Git e GitHub


---

## Funcionalidades da página

- Página de apresentação profissional
- Layout moderno e responsivo
- Avatar com bordas arredondadas
- Texto justificado para melhor leitura
- Ícones sociais clicáveis
- Estrutura simples para fácil manutenção

---

## Seções da página

- Cabeçalho com nome e descrição profissional
- Apresentação pessoal
- Lista de habilidades
- Links para redes profissionais
- Rodapé com ícones sociais
